## Technical Direction

Because the repository is empty and external credentials may not be available, the most reliable MVP is a dependency-light browser app:

- `index.html`, `styles.css`, and `src/main.js`.
- Browser camera/microphone capture to represent LiveKit-style live capture in the demo.
- A mock model adapter returning the same JSON shape expected from the model service.
- Timeline stored in browser memory/localStorage for demo continuity.
- Agent responses generated locally from the timeline, with a clean seam for replacing with Gemini.
- Health alerts generated by deterministic rules from the timeline.
